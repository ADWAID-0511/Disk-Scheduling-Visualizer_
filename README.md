💽 Disk Scheduling Visualizer

An interactive web-based tool to visualize and compare disk scheduling algorithms with real-time animation and performance metrics.

🌐 Live Demo

👉 (https://adwaid-0511.github.io/Disk-Scheduling-Visualizer_/)



✨ Features

📊 Real-time disk head movement visualization
⚡ Supports multiple algorithms:

   FCFS (First Come First Serve)
   SSTF (Shortest Seek Time First)
   SCAN (Elevator Algorithm)
   C-SCAN (Circular SCAN)
📈 Graph-based visualization (Chart.js)
📉 Performance comparison across algorithms
🧠 Automatic best algorithm detection



📊 Output Displays

 ✔ Seek Sequence
 ✔ Total Seek Time
 ✔ Number of Head Movements
 ✔ Algorithm Comparison Table
 ✔ Best Algorithm Indicator


⚙️ Algorithms Overview

🔹 FCFS (First Come First Serve)

Processes requests in the order they arrive. Simple but inefficient for large datasets.

🔹 SSTF (Shortest Seek Time First)

Selects the closest request to minimize seek time, but may lead to starvation.

🔹 SCAN (Elevator Algorithm)

Moves the disk arm in one direction, servicing requests, then reverses direction.

🔹 C-SCAN (Circular SCAN)

Moves in one direction and jumps back to the beginning, ensuring uniform wait time.

🎯 Objective

To analyze and compare disk scheduling algorithms based on performance metrics like seek time, efficiency, and movement optimization.
